Es gibt viele verschiedene Möglichkeiten der Objektanimation mit Hilfe von HTML5 & CSS3

Man kann Objekte beliebig drehen.




transform: rotate(30deg);

Verschiedene Ebenen verwenden um große Dinge

...und kleine Dinge darzustellen.

Man kann Dinge beliebig positionieren


transform: translateY(-25px);

und sie je nach belieben

skalieren


transform: scale(0.7);

Außerdem kann man...

natürlich auch Objekte in 3D animieren!

Transitions


  • kann bei Zustandsänderungen definiert werden (z.B. Hover)
  • früher:
    • Änderungen bspw. bei Hover werden direkt ausgeführt
    • Änderungen über Zeit oder zeitverzögert nicht alleine durch CSS möglich
  • heute:
    • Möglichkeit der Änderung über Zeit
    • Zeitverzögerungen möglich

Transitions Beispiel

Hier kommt die Transition zum Einsatz

<div id="transitionsex">
 <p>Hier kommt die Transition zum Einsatz</p>
</div>

#transitionsex p {
 font-size:22px;
 color:white;
 background-color:black;
 padding-left:0px;
 -webkit-transition: all 2s ease;
 transition: all 2s ease;
}
#transitionsex p:hover {
 font-size:26px;
 font-weight:bold;
 color:#00ff05;
 padding-left:170px;
}

Keyframes

  • Werden für Animationen benötigt
  • Genaue Abstufung einzelner Animationsschritte möglich

Keyframes Beispiel

Keyframes Beispiel


<div id="keyframecontainer">
 <div id="keyframesex"></div>
</div>

@keyframes keyframesex {
 0% { top:0; left:0;}
 25% { top:0; left: 530px;}
 50% { top: 330px; left: 530px;}
 75% { top:330px; left:0;}
 100% { top:0; left: 0;}
}
#keyframesex {
 background-color:black;
 width:50px;
 height:50px;
 position:relative;
 top:0;
 left:0;
 animation-name: keyframesex;
 animation-duration: 4s;
 animation-iteration-count: infinite;
 animation-timing-function: linear;
}
#keyframecontainer {
 position:relative;
 width: 600px;
 height:400px;
}

Unterschiede zu JavaScript

  • JS-Animationen sehen überall gleich aus
  • Programmierung von Animationen aufwändiger (sofern kein Framework benutzt wird)
  • CSS3-Animationen sind performanter
  • nicht jeder Browser unterstützt CSS3

Übungsaufgabe 1


  • Basic
    • Fährt man mit der Maus über den Bildausschnitt, so soll mit ihm folgendes passieren: Skalierung auf doppelte Grüße, Drehung im Uhrzeigersinn um 10° und Neigung nach hinten um 70°.
  • Advanced
    • Nun soll sich der Bildausschnitt auch bewegen. Liegt die Maus darüber, dreht sich der Ausschnitt permanent im Uhrzeigersinn. Gleichzeitig soll er sich bis zur Hälfte des Weges um 70° nach hinten und bis zur Vollendung seiner Runde wieder vollständig nach vorne geneigt haben. Zusätzlich wird der Bildausschnitt auch hier auf zweifache Größ skaliert.

Übungsaufgabe 2


  • Basic
    • Jeder der fünf Flummis soll, wenn die Maus ihn überfährt, anfangen zu hüpfen.
  • Advanced
    • Nun sollen die Bälle selbstständig springen (in unseren Fall beginnen sie zu hüpfen, wenn die Seite geladen ist). Während der erste Ball fünfmal in die Luft springt, tut es der zweite nurnoch viermal, der dritte dreimal usw. Nice to have: wer schafft es, die Bälle im Abstand von einer Sekunde nacheinander loshüpfen zu lassen?
