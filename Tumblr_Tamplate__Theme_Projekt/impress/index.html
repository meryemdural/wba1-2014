impress.js Template

Dieses Template beinhaltet grundlegende Styles für eine Präsentation auf Basis von impress.js.

Jede Slide kann natürlich nach belieben durch Styles ergänzt werden. Im Folgenden wird näher auf die Verwendung der vordefinierten Klassen eingegangen.

Slides

Eine Slide ist eine einfache Präsentationsfolie. Sie besitzt neben der Standardklasse step ebenfalls die Klasse slide.

Eine Slide fügt einem normalen Step, wie die vorige Folie einer ist, einige Eigenschaften hinzu:

  • Sie besitzt eine feste Breite und Höhe.
  • Der Inhalt ist nicht vertikal mittig ausgerichtet.
  • Sie besitzt einen grauen Rahmen.

Die meisten Klassen sind nach der Idee des objektorientierten CSS gestaltet. Grundlegende Gestaltungen können durch einfaches hinzufügen von Klassen in HTML erreicht werden.

Man muss nicht zwingend eigenes CSS hinzufügen.*

* Beispielsweise besitzt der große Text die Klasse big. Dieser kleinere Text ist hingegen mit der Klasse small ausgezeichnet.

Überschriften

müssen nicht unbedingt farbig hinterlegt sein

Standardmäßig sind sie sogar gar nicht farbig hinterlegt. Für den farbigen Hintergrund sorgt die Klasse color-block.

Überschriften fangen hier übrigens mit h2 an, nicht h1.

Standardmäßig

besitzt eine Überschrift auch keinen Abstand nach unten. Somit kann man mehrere Überschriftenblöcke untereinander anordnen.

Der Abstand nach unten …

Farben

Im Text kann man auch farbig schreiben.
Das machen die Klassen pink, green, brown und blue.

Hängt man diesem Klassennamen ein -color-block an, so wird der Text (wie bei den Überschriften) farbig hinterlegt.

Man sollte beachten, dass Verlinkungen standardmäßig auch pink hinterlegt werden.

Manchmal möchte man vielleicht Code schreiben. Dafür verwendet man <pre> in Verbindung mit <code>.

<pre><code>
    <!-- Code hier einfügen -->
</code></pre>
def display_my_syntax(awesome)
    return true
end
